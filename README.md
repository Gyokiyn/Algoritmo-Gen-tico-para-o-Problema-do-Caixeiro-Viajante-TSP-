# Algoritmo Genético para o Problema do Caixeiro Viajante (TSP)

## Descrição do Projeto

O problema do Caixeiro Viajante (Traveling Salesman Problem - TSP), um problema clássico de otimização combinatória. O objetivo do TSP é encontrar o caminho de menor custo que passa por todas as cidades, retornando à cidade de origem. Este problema é conhecido por ser NP-difícil, o que significa que soluções ótimas são de difícil obtenção para grandes instâncias. Por isso, heurísticas como os algoritmos genéticos são frequentemente utilizadas para encontrar boas soluções aproximadas.
